okCycle

Tiny, modular, flexible slideshow

Demonstration

Fading

          
$("#example_one").okCycle({
  autoplay: true,
  effect:'fade', ui: ['caption','navigation','pagination','currentPage'],
  afterSetup:function(){
    var slideshow = this;
    this.hover(function(){ slideshow.pause(); },function(){ slideshow.play(); });
  }
});

Scrolling

            
$("#example_two").okCycle({
  effect:'scroll',
  ui: ['caption','navigation','pagination','currentPage']
});

The normal behavior is to use the elements title attribute to provide captions. However, If you give your slide a title prefixed with an octothorpe, it will use the innerHTML from an element with an id of the same value.
